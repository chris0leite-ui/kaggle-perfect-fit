# The Perfect Fit — Kaggle Competition

https://www.kaggle.com/competitions/the-perfect-fit

Tabular regression competition.

## Setup

```bash
pip install -r requirements.txt
```

## Workflow (Red-Green TDD)

1. Write a failing test in `tests/`
2. Implement the minimum code in `src/` to make it pass
3. Refactor, then repeat

## Data

Download competition files from Kaggle and place them in `data/`.
